# Kak Strelyatb

Простой однопользовательский VR-симулятор тира на Unity.

## Описание игровых механик и управления

- Нажатием на правый триггер геймпада игрок совершает выстрел;
- Нажатием на RB геймпада игрок взаимодействует с предметами;
- Левым стиком геймпада игрок управляет положением руки в пространстве;
- Правым стиком геймпада игрок управляет поворотом руки в пространстве;
- Крестовиной геймпада игрок перемещается по карте;
- Мишень меняет цвет/разрушается при попадании.

## Визуал

1. Общий стиль: Игра выполнена в стиле "Симуляция" с преимущественными цветами окружения белый, серый, голубой. Освещение сбалансированное.
2. Объекты и текстуры: Пространство состоит из плоскости, по которой передвигается игрок. На территории пространства в его середине расположен стол с пистолетом. На краю плоскости стоит стена с мишенями.
3. Пользовательский интерфейс (UI): Минималистичный интерфейс включает полупрозрачный таймер обратного отсчета до конца игры.
4. Эффекты: Звуковые и визуальные эффекты стрельбы.
